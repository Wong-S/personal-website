Projects

Optium Time

Project Description:

A full-stack Python Flask web application created during a 5-week period for my capstone project at Hackbright Academy. This app allows the user to wind down before bed by journaling and setting a Moment.js alarm clock. The user also sees a graphical display of their logged total hours slept per week/month and NREM/REM stages using Chart.js. The Youtube Data API allows the user to query and search for videos with specific ambient sounds to act as an aide for their sleep routine. The user can use the app’s Twilio API feature to notify the user’s total sleep cycles from that night.

Tech Stack: Python, PostgreSQL, Javascript, Flask, HTML, CSS, AJAX, Bootstrap, Charts.js, Moment.js

APIs: Youtube Data API, Twilio API

Picasso's Pizzeria

Project Description:

Winner of the Hackbright's Phenomenal Woman Scholarship in Fall 2020. This is a build-your-own terminal-based pizza game created within a two week period. The story and animation was created using Python 3 and Turtle enhanced graphics. The user can select topping items from a formatted menu, as well as add, edit, and delete topping selections before checking out to pay.

Tech Stack: Python, Turtle graphics

Goal Tracker App

Project Description:

A collaborative team project that was created during Mintbean.io January 2021 hackathon to learn and gain exposure to React.js.
