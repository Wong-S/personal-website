About Me

Howdy! I am a former biological scientist transitioning into Software Engineering. Born and raised in Texas, I started my self-learning programming journey when Covid hit. At the time, I was not satisfied with where my life was and when the world seemed to turn upside down, my friend suggested I look into programming due to the exciting growth and opportunities. I was hesitant, but decided to do some self-reflection and self-learning to try something new in addition to my science background. Since then, I realized how enjoyable building and creating a project was from (sometimes) just a few lines of code. The journey has not all been smooth sailing, but I am embracing every obstacle, language, and bug that comes my way. Sometimes I cannot believe that I attended an accelerated software engineering fellowship at Hackbright Academy for 12 weeks and graduated this past December 2020. My programming journey has only just begun and I cannot wait to see where I will be in the future. Please feel free to connect with me on social media or contact me below for more information.

Projects

Optium Time

A full-stack Python Flask web application created during a 5-week period for my capstone project at Hackbright. This app allows the user to wind down before bed by journaling and setting a Moment.js alarm clock. The user also sees a graphical display of their logged total hours slept per week/month and NREM/REM stages using Chart.js. The Youtube Data API allows the user to query and search for videos with specific ambient sounds to act as an aide for their sleep routine. The user can use the app’s Twilio API feature to notify the user’s total sleep cycles from that night.

Picasso's Pizzeria

A build-your-own terminal-based pizza game created within a two week period. The story and animation was created using Python 3 and Turtle graphics. Selected winner for the Hackbright's Women Scholarship.
